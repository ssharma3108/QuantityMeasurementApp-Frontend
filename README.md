# 📦 Quantity Measurement App (Frontend - Angular)

## 🚀 Overview
The **Quantity Measurement App (Frontend)** is an Angular-based web application that allows users to perform unit and quantity conversions efficiently. It provides a clean and responsive UI with real-time conversion and seamless integration with backend APIs.

---

## 🧩 Features
- 📏 Convert between multiple measurement units  
- ⚡ Fast and dynamic UI using Angular  
- 🔄 Real-time conversion updates  
- 🧠 Easy-to-use interface  
- 🌐 REST API integration  
- 📱 Responsive design for all devices  

---

## 🛠️ Tech Stack
- **Framework:** Angular  
- **Languages:** TypeScript, HTML, CSS  
- **Styling:** CSS / Bootstrap *(if used)*  
- **HTTP Client:** Angular HttpClient
